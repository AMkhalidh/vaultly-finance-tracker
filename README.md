# Vaultly — Personal Finance Tracker

A full-stack personal finance tracker built with React, Node.js, Express and MongoDB.

## Features
- User authentication (register/login with JWT)
- Add, edit, delete income & expense transactions
- Dashboard with live stats: balance, income, expenses, savings rate
- 6-month trend chart and spending breakdown by category
- Monthly budget tracker with progress bar
- Filter transactions by month, year, type and category
- Dark mode UI

## Tech Stack
**Frontend:** React, React Router, Recharts, Axios
**Backend:** Node.js, Express, MongoDB, Mongoose, JWT, bcryptjs
